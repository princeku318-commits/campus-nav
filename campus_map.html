📍 Location Access Needed

Your browser blocked GPS access. This usually happens when the file is opened directly (file://) or permission was denied.

  • 1Serve over HTTP/HTTPS — open via VS Code Live Server, Python's http.server, or upload to any web host. GPS does not work on file://
  • 2Android Chrome: Tap the lock icon in the address bar → Site settings → Location → Allow
  • 3iPhone / iPad: Go to Settings → Safari → Location → Allow (or "Ask Next Time")
  • 4Reload the page and tap the 📍 button again

Without GPS, routes will be drawn from the Main Gate as a fallback.
